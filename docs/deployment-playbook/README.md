# AI PMO 導入プレイブック

> クライアントのプロジェクトに AI PMO を導入するための **全工程ステップバイステップ手順書**。
> コンサルタント・PMO 担当・営業のいずれにも使える共通リファレンス。
> 顧客提案資料の素材としても使える（要編集）。

## 全体マップ

```
┌────────────────┐  ┌────────────────┐  ┌──────────────┐  ┌────────────┐  ┌──────────────┐
│ Phase 1        │→ │ Phase 2        │→ │ Phase 3      │→ │ Phase 4    │→ │ Phase 5      │
│ Discovery      │  │ 環境セットアップ │  │ データ統合    │  │ ローンチ    │  │ 運用         │
│ (1〜2週間)      │  │ (1週間)         │  │ (2〜3週間)    │  │ (1週間)     │  │ (継続)        │
└────────────────┘  └────────────────┘  └──────────────┘  └────────────┘  └──────────────┘
        ↑                  ↑                  ↑               ↑                ↑
      人間 100%         95% 自動           半自動            AI 主導         AI + 監視
```

合計: **PoC 開始まで 5〜7週間** / **PoC 期間 3ヶ月** が標準。
