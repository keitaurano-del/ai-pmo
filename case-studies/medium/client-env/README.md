# client-env/ — 仮想クライアント環境

NEXUS-RTL プロジェクトにおいて AI PMO が運用する成果物群。

実運用では Confluence / Box / Slack 等に分散するが、本検証環境では単一リポ内にフラットに配置して、構造と中身の見え方を可視化する。

## ディレクトリと役割

| ディレクトリ | 内容 | 主担当エージェント | 更新頻度 |
|--------------|------|----------------------|----------|
| `00-context/` | ステークホルダー一覧、用語集 | — (初期固定) | 月次見直し |
| `01-charter/` | プロジェクト憲章、スコープ定義 | — (人間主導) | キックオフ時+変更時 |
| `02-wbs/` | WBS（Markdown + CSV） | レポーター | 週次更新 |
| `03-schedule/` | マスタースケジュール、マイルストーン | レポーター | 週次更新 |
| `04-meetings/` | 議事録 | スクライブ | 会議のたび |
| `05-issues/` | 課題管理表 | ウォッチャー + トリアージャー | 日次更新 |
| `06-risks/` | リスク台帳 | ウォッチャー（将来：リスクアナリスト） | 週次見直し |
| `07-reports/` | 週次・月次報告書 | レポーター | 週次/月次 |
| `08-decisions/` | 意思決定ログ | スクライブ + トリアージャー | 発生時 |

## ファイル末尾の出所表記

AI PMO が生成・更新したファイルは末尾に以下を記載：

```
---
Generated by: <AgentName> (AI PMO)
Last updated: 2026-MM-DD HH:MM
Reviewed by: <human PMO name>
```

人間が直接書いたファイル（PJ 憲章、組織図など）はこのフッターを持たない。

## 基準日

本環境のスナップショットは **2026-05-15** 時点（PoC 開始から第7週、12週中）。
